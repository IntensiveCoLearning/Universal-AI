-   安装 MetaMask 钱包

-   添加 ZetaChain 网络到 MetaMask
    
-   获取测试网代币
    

### 2\. AI 开发环境

-   注册 Qwen 账号
    
-   申请 API Key
    
-   测试 API 调用

-   安装 Node.js（推荐 v18+）

-   安装 Git
    
-   准备开发工具：

    -   ethers.js / web3.js（区块链交互）
