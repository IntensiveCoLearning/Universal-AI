安装zetachain

```
npm install -g zetachain@latest
```

命令

启动localnet

```
zetachain localnet start
```

遇到错误（环境ubuntu22）

```
bigint: Failed to load bindings, pure JS will be used (try npm run rebuild?)
[LOCALNET] Error: 'anvil' not found. Please install Foundry: https://getfoundry.sh
```

解决方法

```
# 1. 安装 foundryup（Foundry 的安装器）
curl -L https://foundry.paradigm.xyz | bash

# 2. 把 foundryup 加入 PATH（如果尚未生效）
export PATH="$HOME/.foundry/bin:$PATH"

# 3. 使用 foundryup 安装最新版 Foundry 工具
foundryup
```

再次执行启动命令

```
zetachain localnet start
```

查询跨链余额

```
zetachain query balances --evm address
```
