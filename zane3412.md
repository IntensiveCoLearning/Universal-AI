# 2025-11-25

创建项目

```
npx zetachain@latest new --project hello
```

版本： zetachain@7.4.0

安装包

```
npm install
```

新配置并更新由Foundry的Soldeer管理的Solidity依赖项，确保您的合约是针对外部库的最新兼容版本构建的。

```
forge soldeer update
```

编译合约命令

```
forge build
```

启动本地localnet

```
npx zetachain localnet start
```

### 部署到本地localnet

部署合约

获取私钥

```
PRIVATE_KEY=$(jq -r '.private_keys[0]' ~/.zetachain/localnet/anvil.json) && echo $PRIVATE_KEY
```

部署通用合约

```
UNIVERSAL=$(forge create Universal \
  --rpc-url http://localhost:8545 \
  --private-key $PRIVATE_KEY \
  --evm-version paris \
  --broadcast \
  --json | jq -r .deployedTo) && echo $UNIVERSAL
```

调用合约

获取gateway

```
GATEWAY_EVM=$(jq -r '.["11155112"].contracts[] | select(.contractType == "gateway") | .address' ~/.zetachain/localnet/registry.json) && echo $GATEWAY_EVM
```

通过gateway调用call方法

```
npx zetachain evm call \
  --rpc http://localhost:8545 \
  --gateway $GATEWAY_EVM \
  --receiver $UNIVERSAL \
  --private-key $PRIVATE_KEY \
  --types string \
  --values hello
```

### 部署到测试网

设置钱包本地私钥

```
PRIVATE_KEY=私钥  && echo $PRIVATE_KEY
```

部署测试链

```
UNIVERSAL=$(forge create Universal \
  --rpc-url https://zetachain-athens-evm.blockpi.network/v1/rpc/public \
  --private-key $PRIVATE_KEY \
  --broadcast \
  --json | jq -r .deployedTo)
```

从base链调用通用合约

```
npx zetachain evm call \
  --chain-id 84532 \
  --receiver $UNIVERSAL \
  --private-key $PRIVATE_KEY \
  --types string \
  --values hello
```

# 2025-11-24
<!-- DAILY_CHECKIN_2025-11-24_START -->


安装zetachain

```
npm install -g zetachain@latest
```

命令

启动localnet

```
zetachain localnet start
```

遇到错误（环境ubuntu22）

```
bigint: Failed to load bindings, pure JS will be used (try npm run rebuild?)
[LOCALNET] Error: 'anvil' not found. Please install Foundry: https://getfoundry.sh
```

解决方法

```
# 1. 安装 foundryup（Foundry 的安装器）
curl -L https://foundry.paradigm.xyz | bash

# 2. 把 foundryup 加入 PATH（如果尚未生效）
export PATH="$HOME/.foundry/bin:$PATH"

# 3. 使用 foundryup 安装最新版 Foundry 工具
foundryup
```

再次执行启动命令

```
zetachain localnet start
```

查询跨链余额

```
zetachain query balances --evm address
```
